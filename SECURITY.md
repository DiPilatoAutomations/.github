# Security Policy

We build systems that answer phones and handle customer data for real businesses. Security reports get taken seriously and handled fast.

## Reporting a vulnerability

Email **jondipilato@dipilatoautomations.com** with:

- What you found and where (repo, endpoint, or product)
- Steps to reproduce it
- What an attacker could do with it

You'll get a response within **2 business days**. Please don't open a public issue for security problems, and give us a chance to fix it before you talk about it publicly.

## Scope

All public repositories under this organization, plus the products we host (Job Rescue, BookingBot, Job Estimate AI).
